# Рынок

## Создание

Поставь бочку и напиши команду:

```
/shop create
```

Поставь шалкер берюзового цвета. Это будет склад для пополнения товара и выручки.

{% hint style="info" %}
 Склады синхронизируются, их можно разместить где угодно и сколько угодно.
{% endhint %}
